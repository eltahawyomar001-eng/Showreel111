## Animation Performance

### What Makes It Fast

1. **Hardware Acceleration**
   - Uses `transform` and `opacity` only
   - No layout recalculations
   - GPU-accelerated rendering

2. **Framer Motion Optimization**
   - Automatic will-change management
   - Layout projection for smooth morphs
   - Batched updates

3. **Scroll Performance**
   - `useScroll` is throttled internally
   - Transform calculations are memoized
   - No scroll event listeners (uses IntersectionObserver)

4. **Code Splitting**
   - Next.js automatic code splitting
   - Components lazy-loaded
   - Minimal initial bundle
